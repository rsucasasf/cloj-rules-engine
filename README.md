# cloj-rules-engine

**cloj-rules-engine** is a rules engine written in Clojure and designed to work with Java.

## Prerequisites

1. Java version 8

2. [Leiningen][] 2.0.0 or above installed.

[leiningen]: https://github.com/technomancy/leiningen
